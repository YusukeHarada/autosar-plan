## 3. 機能要件

### 3.1 要件記述ガイドライン

本書では以下のキーワードを用いて要件の強度を示す。

| キーワード | 意味 |
|---|---|
| **SHALL** | 絶対的要件。必ず実装すること |
| **SHALL NOT** | 絶対的禁止事項 |
| **SHOULD** | 推奨要件。技術的理由がある場合のみ逸脱可（逸脱理由を文書化すること） |
| **MAY** | 任意要件。実装は委託先の判断でよい |

### 3.2 SWC要件記述形式

各SWCは以下の形式で要件を記述する。

```
SWC-ID   ：[例：SWC-HYD-001]
SWC名称  ：[機能を表す名称]
機能概要 ：[SWCが実行する処理の概要（2〜3文）]
入力ポート：[シグナル名 / データ型 / 更新周期 / 接続元SWC or BSWモジュール]
出力ポート：[シグナル名 / データ型 / 更新周期 / 接続先SWC or BSWモジュール]
処理要件 ：[SHALL/SHOULD/MAYを用いた処理ロジックの要件]
安全要件 ：[フォールバック動作・安全状態遷移に関する要件]
```

### 3.3 サンプル：エンジン回転数受信SWC

```
SWC-ID   ：SWC-HYD-001
SWC名称  ：EngineSpeedReceiver
機能概要 ：エンジン制御ECU（ECM）からJ1939経由で受信したエンジン回転数シグナル
           を受け取り、有効性チェックを行ったうえで内部バスに配信する。

入力ポート：
  - ポート名       ：in_J1939_EngineSpeed
  - シグナル名     ：EngineSpeed（SPN 190）
  - データ型       ：uint16（分解能0.125rpm/bit、オフセット0）
  - 更新周期       ：100ms（PGN 61444 EEC1 の受信周期）
  - 接続元         ：Com（CAN受信バッファ）

出力ポート：
  - ポート名       ：out_EngineSpeed_rpm
  - シグナル名     ：EngineSpeed_rpm
  - データ型       ：float32（単位：rpm）
  - 更新周期       ：100ms
  - 接続先         ：SWC-HYD-003（油圧指令算出SWC）

処理要件：
  [SRS-HYD-001-01] 本SWCは SHALL、受信シグナルの値域（0〜3000rpm）チェックを
                   実施し、範囲外の場合はシグナル無効フラグを立てること。
  [SRS-HYD-001-02] 本SWCは SHALL、100ms以上シグナルを受信しない場合（タイムアウト）
                   を通信途絶として検出し、Demに故障イベントを通知すること。
  [SRS-HYD-001-03] 本SWCは SHOULD、ローパスフィルタ（カットオフ周波数：要確定）を
                   適用し、ノイズを除去した値を出力すること。

安全要件：
  [SRS-HYD-001-S01] タイムアウトまたは値域エラー検出時、本SWCは SHALL、
                    フォールバック値（0rpm）を出力し、SafetyManager SWCに
                    安全イベントを通知すること。
```

### 3.4 サンプル：電磁弁指令SWC

```
SWC-ID   ：SWC-HYD-004
SWC名称  ：SolenoidValveCommander
機能概要 ：油圧指令算出SWCから受け取った指令値をPWMデューティ比に変換し、
           電磁弁ドライバICへ出力する。出力前に上下限リミット処理を行う。

入力ポート：
  - ポート名       ：in_ValveCmd_pct
  - シグナル名     ：SolenoidCmd_percent
  - データ型       ：float32（単位：%、範囲：0.0〜100.0）
  - 更新周期       ：10ms
  - 接続元         ：SWC-HYD-003（油圧指令算出SWC）

出力ポート：
  - ポート名       ：out_PWM_DutyCycle
  - シグナル名     ：PWM_Duty
  - データ型       ：uint16（分解能0.1%/bit）
  - 更新周期       ：10ms
  - 接続先         ：IoHwAb（PWM出力ドライバ）

処理要件：
  [SRS-HYD-004-01] 本SWCは SHALL、入力指令値を0〜100%の範囲にクランプしたうえで
                   PWMデューティ比に変換すること。
  [SRS-HYD-004-02] 本SWCは SHALL、指令値の変化率（スルーレート）を
                   [xx %/ms — 要確定] 以下に制限すること。
  [SRS-HYD-004-03] 本SWCは SHALL NOT、SafetyManagerから安全停止要求を受けている
                   間、ゼロ以外のPWMデューティを出力すること。

安全要件：
  [SRS-HYD-004-S01] 安全停止要求受信時、本SWCは SHALL、10ms以内にデューティ0%
                    を出力し、電磁弁を非励磁状態（フェールセーフ方向）にすること。
```

## 7. 変更管理・トレーサビリティ

### 7.1 要件IDの付け方

```
SRS - [システム略称] - [連番3桁]

例：
  SRS-HYD-001   : 油圧制御ECU（HYD）の要件001番
  SRS-HYD-001-01: SWC-HYD-001に属する機能要件01番
  SRS-HYD-001-S01: SWC-HYD-001に属する安全要件01番
  SRS-NFR-TIM-001: 非機能要件（NFR）タイミング（TIM）001番
```

> 要件IDは一度付与したら変更しないこと。廃止する場合はステータスを「廃止」とし、IDは欠番としてそのまま残すこと。
